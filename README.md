常用工具集合，通过 `yt` 入口掌控一切

## 总体规划

- 通过 `clap` 实现命令行交互方式
- 通过 `tauri` 实现 UI 交互
- 提供 Web 在线服务

## 使用方式

所有的子命令调用都由 `yt` 发起，例如：`yt timestamp 1752915718`

## 已支持工具介绍

- timestamp
